# Biquadris--

A two-player, Tetris-like game implemented in C++.
This project focuses on object-oriented design, game-state management,
and modular architecture.

## Features
- Two-player competitive gameplay
- Extensible block and rule system
- Command-driven interface
- Support both TUI and GUI

This repository is shared for demonstration and learning purposes.
